# Case Study: ML Coating Voorraad

## Problem
Floor operations needed a fast way to track powder inventory without heavy ERP friction.

## Solution
A focused web app built around speed of entry, clear stock visibility, and quick consumption updates.

## Outcome Goals
- Reduce time to register stock changes
- Improve visibility of low-stock materials
- Reduce manual tracking errors

## Technical Notes
- Next.js + TypeScript frontend
- Domain model for boxes/weight/partial consumption
- Undo support for recent inventory actions
